## Project Overview
Proyek ini menganalisis tren polusi udara di kota-kota besar India dan membangun model **Deep Learning (LSTM)** untuk memprediksi indeks kualitas udara (AQI). Ini adalah bagian dari portofolio teknologi saya untuk mendukung **SDG 11: Sustainable Cities and Communities**.

##  Key Features
- **Exploratory Data Analysis (EDA)**: Menemukan korelasi kritis antara polutan PM2.5 dan tingkat AQI.
- **Geospatial Mapping**: Peta interaktif persebaran polusi menggunakan Folium.
- **AI Forecasting**: Prediksi tren kualitas udara masa depan menggunakan **TensorFlow**.

## Tech Stack
- **Languages:** Python
- **Libraries:** TensorFlow, Keras, Pandas, Folium, Seaborn, Matplotlib.
- **Tools:** Kaggle Notebook, GitHub.
